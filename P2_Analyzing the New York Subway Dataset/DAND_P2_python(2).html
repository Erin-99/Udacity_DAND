## 是否生还与年龄的相关性，用survived代表是否生还的结果（因变量y），age代表年龄（自变量x）
def correlation(x, y):

    std_x = (x-x.values.mean())/x.std(ddof=0)
    std_y = (y-y.values.mean())/y.std(ddof=0)
    
    return (std_x*std_y).mean()
survived = titanic_data['Survived']
age = titanic_data['Adjusted_age']
correlation(age,survived)

3、初步结论¶

## 用线性回归模型来计算一下是否生还与年龄、性别、车票费用、车厢等级Pclass的关系。
import pandas as pd
import statsmodels.api as sm
y = survived
sex = (titanic_data['Sex']=='male').astype('int')
pclass = pd.get_dummies(titanic_data['Pclass'],prefix='Pclass')
fare = titanic_data['Fare']
X = pd.concat([pclass,age,sex,fare],axis=1)
X=sm.add_constant(X)
est=sm.OLS(y,X)
est=est.fit()
est.summary()

4、思考及后续探索方向¶

以上是关于titanic数据的初步探索，其中age缺失值用了中值median填充，从本次的结果来看，用中值填充使得数据更集中，并不能确定是否是个有效的方法，需要在后续的探索中验证。另外，此数据集只有891条记录，不到总数据集的一半，年龄中有接近20%数据缺失，虽然用中值填充，却也很难保证它的合理性，更不能保证由此得出的结论的正确性。而且此次探索只涉及了年龄、性别、Pclass等级、费用这四个方面，得出的Rsqure只有0.38，可见并不能很好的解释结果，其他因素尚未进行探索，如兄弟姐妹个数，Embarked，以及一些未进行记录的参数，如身体状况等，都有可能影响最终的结果。
